# GitList
List repositories using Github public API

- [x] Repository List
- [x] Infinite scroll
- [x] Pull to refresh
- [x] Unit test
- [x] View Code
- [x] Async image loading
- [x] Slather with Fastlane
- [x] Swift 4, Xcode 10
- [x] Cocoapods

- To run, pod install, open itList.xcworkspace
- To run Slather, fastlane coverage, results are in slather_result
